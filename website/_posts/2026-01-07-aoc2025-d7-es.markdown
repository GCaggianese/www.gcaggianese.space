# Puzzle [Dia 7: Laboratories](https://adventofcode.com/2025/day/7)

## Parte uno

Procesar de arriba hacia abajo. Para cada celda `a_{i,j}`:
- `.` -> no hacer nada
- `S` o `|` -> mirar $a_{i+1,j}$ (celda inferior):
  - si es `.` -> cambiarla a `|`
  - si es `^` -> $a_{i+1,j-1}$ y $a_{i+1,j+1}$ pasan a ser `|`
    - **Importante**: marcar `^` como usado

Contar cuántos splitters `^` fueron marcados como usados.

## Parte dos

Contar todas las líneas temporales posibles cuando los splits `^` van
en **una dirección por vez**.

Para cada celda `a_{i,j}`:
- `col >= cols` -> devolver 1 (camino completado)
- `row >= rows` -> devolver 1 (llegó al fondo)
- `^` en $a_{r,col}$ -> recursión:
  - `countPaths(r+1, col-1)` + `countPaths(r+1, col+1)`
  - si `col == 0` -> la rama izquierda devuelve 1 (fuera de grilla = completado)

### Ejemplo

```
.S.        .S.   .S.
...   →    .|.   .|.
.^.        |^.   .^|
...        |..   ..|
..^        |.^   .|^
```

Camino izquierdo: 1 (llega al fondo)
Camino derecho: toca `^` en (4,2) -> se divide de nuevo -> 2
**Total: 3 líneas temporales**

### Optimización

Memoización con `HashMap(State, usize)` donde `State = {row, col}`.
Evita recalcular caminos desde la misma posición.
