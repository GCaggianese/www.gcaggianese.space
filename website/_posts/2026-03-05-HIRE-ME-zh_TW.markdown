## 基礎架構與 SysAdmin 顧問

現代部署不應該令人頭痛。我專門把原始伺服器轉換成安全、自動化、可投入
生產的環境。

* **客製 KVM 與 VPS 佈建：** 端到端設定伺服器環境。
* **容器編排：** Docker 與 Docker Compose 架構設計與除錯。
* **Web 伺服器與路由：** Nginx 反向代理設定、DNS 管理與 SSL/TLS 實作。
* **這個網站就是證明：** 這個網站本身就是實際案例。我親自管理 DNS，把它託管在客製 KVM 上，透過反向代理路由流量，並使用我自己寫的 Org-mode 到 Jekyll parser 發佈內容。
