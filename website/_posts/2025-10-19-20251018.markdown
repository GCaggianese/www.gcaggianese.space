# 從……往…… (表示方向)/ V + 起來 各五個句子

## 從……往……

-   第一：銀行 → 郵局
    從銀行往前走三個路口，在第三個路口左轉然後走一半路，在這裡右轉就你可以看到郵局。

![銀行 → 郵局](./../assets/Notes_251019_003746_c6f.jpg)

-   第二：超市 → 學校
    從超市往前走一個路口，然後左轉和走一個路口，在這裡你會看到一個紅綠燈，看到這個以後，右轉就走過兩個路口。最後左轉就你會看到學校。

![超市 → 學校](../assets/map.jpg)

-   第三：白如玉的家 → 捷運站
    從白如玉的家往前走，看到紅綠燈時，在到達路口前左轉，然後你會看到捷運站。

![白如玉的家 → 捷運站](../assets/Notes_251019_003802_135.jpg)

-   第四：火車站 → 超市
    從火車站往前走一個路口，在這個路口左轉，走兩個路口，過第二個路口，看到紅綠燈時，在到達路口前右轉，然後你可以看到超市。

![火車站 → 超市](../assets/Notes_251019_003754_d3b.jpg)

-   第五：銀行 → 超市 從銀行往前走，過一個紅綠燈後左轉，就可以看到超市。

![銀行 → 超市](../assets/Notes_251019_003758_5df.jpg)

# 造５個對話 一問一答 一點都不….
