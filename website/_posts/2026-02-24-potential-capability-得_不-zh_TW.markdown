# Function

When 得 or 不 is betwen verb and resultative/directional complement,
makes a pattern to express if the result is feasible or not.

-   我聽得懂中文音樂
-   老師教的這本書新詞很容易，每個人都學得會

# Negation

Verb + 不 + complement

-   這個地方太吵了，我聽不見你說什麼。
-   他在那裡？我想不起來。

# Questions

-   喂，你聽得到嗎？
-   她是誰？你是不是看不見？

# Exercise

1.  A：你要不要看報紙？<br> 
    B：我的中文不太好， <u>我看不懂</u> 中文報紙。（看懂）
2.  A：你怎麼不學功夫？<br> 
    B：太難了，我真的 <u>學不會</u> 。（學會）
3.  A：那幾個外國人說的英文你 <u>聽得懂</u> 嗎？（聽懂）<br>
    B：我學過幾年英文，他們說的我應該 <u>聽不懂</u> 。
4.  A：現在已經三點了，早上六點你 <u>起得來</u> 嗎？（起來） <br>
    B：<u>起得來</u> ！我習慣了。
5.  A：他說什麼？ <br>
    B：音樂太吵了，我 <u>沒聽見</u> 。（聽見）
